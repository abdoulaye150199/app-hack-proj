# SEN-MOOL PROTECT

## Description

SEN-MOOL PROTECT est une application web qui fournit des solutions de sécurité numérique et de protection des données. Le projet module plusieurs solutions innovantes incluant MoolSafe, une application mobile et un tableau de bord d'analyse.

**Mission :** Souveraineté Numérique & Économie Bleue — New Deal Technologique

---

## 🚀 Installation

### Prérequis

- **Node.js** (v16 ou supérieur)
- **npm** ou **yarn**

### Étapes d'installation

1. **Cloner le repository**
```bash
git clone https://github.com/abdoulaye150199/app-hack-proj.git
cd sen-mool-protect
```

2. **Installer les dépendances**
```bash
npm install
```

3. **Démarrer le serveur de développement**
```bash
npm run dev
```

L'application sera disponible à `http://localhost:5173/`

---

## 📦 Dépendances principales
